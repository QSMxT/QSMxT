QSMxT takes you from raw scanner DICOMs to quantitative susceptibility maps with
a single self-contained binary. It auto-discovers phase/magnitude pairs in
[BIDS](https://bids.neuroimaging.io/) datasets, runs the full reconstruction
pipeline with sensible defaults, and writes tidy BIDS derivatives — no
environment to manage, no toolbox sprawl.

**Install** it with a single command:

<Tabs syncKey="os">
  <TabItem label="macOS / Linux">
    ```sh
    curl -fsSL https://raw.githubusercontent.com/QSMxT/QSMxT/main/install.sh | sh
    ```
  </TabItem>
  <TabItem label="Windows">
    ```powershell
    irm https://raw.githubusercontent.com/QSMxT/QSMxT/main/install.ps1 | iex
    ```
  </TabItem>
</Tabs>

See the [installation guide](/QSMxT/getting-started/installation/) for other
options, or skip installing entirely and [try it in your browser](https://qsmbly.neurodesk.org/).

## Coming from QSMxT 8.x?

:::note[QSMxT has been rebuilt from the ground up]
**QSMxT v9 is a complete rewrite in Rust** — a single self-contained binary, no
Python environment to manage. The reconstruction pipeline and DICOM/NIfTI
conversion are fully reimplemented and faster, with a new interactive TUI.

Some workflows from the 8.x line aren't part of v9: anatomical segmentation,
template/group-space building, group statistics, NextQSM, and the web UI. If you
rely on those, **the 8.x version is still available** — see the
[`python-legacy` branch](https://github.com/QSMxT/QSMxT/tree/python-legacy) and the
[8.x releases](https://github.com/QSMxT/QSMxT/releases).

A few command changes: `qsmxt <bids>` is now `qsmxt run <bids>`, and conversion is
`qsmxt dicom-convert`. The easiest way in is `qsmxt tui`.
:::

## Try it in your browser

No install, no data to upload anywhere — **[qsmbly](https://qsmbly.neurodesk.org/)**
runs the very same reconstruction algorithms entirely in your browser. It's the
fastest way to experiment with QSM and get a feel for the methods before
processing a full dataset with QSMxT.

## How do I cite QSMxT?

If QSMxT is useful in your research, please cite:

> Stewart AW, Robinson SD, O'Brien K, et al. "QSMxT: Robust masking and artifact
> reduction for quantitative susceptibility mapping." *Magnetic Resonance in
> Medicine* 87.3 (2022): 1289–1300. [doi.org/10.1002/mrm.29048](https://doi.org/10.1002/mrm.29048)

Every run also writes a `references.txt` listing citations for the specific
methods your data and settings exercised.
